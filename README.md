## LLM & GenAI
This repo explores Large Language Models (LLMs) via the OpenRouter API.
The ipynb file can be found under the notebook folder.

## CONTENT

Installation of dependencies
Set up and authentication with OpenRouter API
Different LLM models (GPT, Claude, Mistral)
Prompt engineering
Parameter tuning (temperature, max tokens)
Error handling and retry logic
Use cases: summarisation, translation, code generation, sentiment analysis, conversational AI...
